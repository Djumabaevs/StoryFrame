# Cyber Runner

## Description
A fast-paced cyberpunk run & gun platformer! Take control of an elite cyber soldier fighting through neon-lit cityscapes filled with deadly robots. Run, jump, dash, and blast your way through procedurally generated levels of increasing difficulty.

## Features
- 🎮 Classic run & gun gameplay inspired by Contra and Metal Slug
- 🤖 Multiple enemy types: Walkers, Flyers, and Tanks
- ⚡ Fast-paced combat with dash ability for dodging
- 🏃 Procedurally generated levels - every playthrough is unique
- 💰 Collect coins for bonus points
- 🎯 Time bonus rewards for speedrunners
- 🔊 Retro-style sound effects
- 🎨 Beautiful cyberpunk neon visuals

## Controls
- **A/D** or **←/→** — Move left/right
- **W/↑/SPACE** — Jump
- **J** or **Left Click** — Shoot
- **K** or **SHIFT** — Dash (invulnerable frames!)
- **ESC** — Pause
